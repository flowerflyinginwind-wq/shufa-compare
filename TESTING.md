# 手动浏览器测试清单

完成自动化测试后，运行 `npm run dev`，按以下步骤在浏览器中验证。

## 步骤 2：布局与上传

- [ ] 页面显示标题「书法临摹对比」
- [ ] 左侧有「原帖」「临摹」两个上传区
- [ ] 未上传时 Canvas 显示占位文字
- [ ] 上传 `public/test-original.svg` 和 `public/test-copy.svg` 后显示缩略图

## 步骤 3：叠加对比

- [ ] 两张图上传后 Canvas 显示叠加效果
- [ ] 「仅原帖」「仅临摹」「叠加」切换正常
- [ ] 透明度滑块可调节临摹层透明度

## 步骤 4：差异高亮

- [ ] 切换到「差异高亮」模式
- [ ] 红色区域标出笔画偏差（测试图应有明显高亮）
- [ ] 敏感度滑块可调节高亮范围

## 步骤 5：对齐调节

- [ ] 平移/缩放/旋转滑块可调整临摹位置
- [ ] 「重置」恢复默认对齐
- [ ] 对齐后叠加与差异效果明显改善
