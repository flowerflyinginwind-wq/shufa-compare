# 书法临摹对比工具

在浏览器中对比书法原帖与临摹作品，支持叠加对照和差异高亮，纯前端运行，无需上传服务器。

## 功能

- **双图上传**：支持 JPG / PNG / WebP / SVG，可拖拽或点击上传
- **画布拖拽对齐**：拖拽平移、滚轮缩放临摹层
- **自动初对齐**：一键估算整页平移/缩放/旋转
- **对齐辅助线**：九宫格 + 十字线，辅助结构对齐
- **导出对比图**：保存当前叠加或差异图为 PNG
- **图像增强**：去阴影、对比度拉伸、中值去噪（可开关）
- **结构差异比对**：Otsu 笔画二值化 + Sobel 边缘检测（默认）
- **历史记录**：本地保存最近 6 次对比，一键恢复图片与参数
- **PWA 离线**：构建后可安装到手机/桌面，无网也能用

## 功能扩展

- **拍照上传**：点击「拍照」调用摄像头（手机支持后置镜头）
- **局部放大镜**：开启后在画布悬停，3 倍放大细看笔画
- **历史记录**：保存/恢复对比，导出时自动存档
- **PWA**：`npm run build` 后部署，浏览器可「安装应用」

## 使用建议

1. **拍照**：尽量垂直俯拍，光线均匀，减少阴影
2. **对齐**：先点「自动初对齐」，再在画布上拖拽/滚轮微调
3. **差异**：对齐后切换到「差异高亮」，调低敏感度可看到更多细节偏差
4. **导出**：满意后点「导出对比图 PNG」保存结果

## 构建部署

```bash
npm run build
```

将 `dist` 目录部署到任意静态托管服务即可。部署后可在浏览器中选择「安装应用」添加到主屏幕（需 HTTPS）。
